核心设计理念

传统的 ReAct Agent 往往反应迅速但缺乏深度（System 1）。本架构采用 Deliberative（深思熟虑） 模式，模仿人类专家的"慢思考"过程，通过显式的 感知-建模-推理-决策 链条，确保投资建议建立在严谨的数据分析和逻辑推演之上。

阶段深度拆解

📡 1. 感知 (Perception)
Input Processing
功能描述

作为智能体的"眼睛和耳朵"。不进行复杂推理，专注于从宽泛的主题中提取结构化的事实数据、关键指标和新闻动态。

🧠 2. 建模 (Modeling)
Context Understanding
功能描述

将零散的感知数据转化为结构化的"内部世界模型"。评估当前的市场周期、情绪水位及核心风险因子。

{ "market_state": "复苏期/过热期...", "economic_cycle": "分析判断...", "risk_factors": ["政策风险", "技术瓶颈"], "opportunity_areas": ["细分赛道A", "出海逻辑"], "market_sentiment": "中性偏乐观" }
💡 3. 推理 (Reasoning)
Divergent Thinking
功能描述

发散思维阶段。基于同一套世界模型，生成多个互斥或并行的投资假设（Scenario Analysis）。例如：乐观剧本 vs 悲观剧本。

⚖️ 4. 决策 (Decision)
Convergent Thinking
功能描述

收敛思维阶段。扮演"投决会主席"角色，评估所有候选方案的风险收益比，拍板选定唯一的执行路径。
