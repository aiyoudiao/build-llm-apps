[改写后的查询]
    客户经理的考核标准是什么？

[查询类型]
    模糊指代型

[答案]
------------------------------------------------------------
客户经理的考核标准分为两大类：**个人业绩考核**和**工作质量考核**（来源：-1 第三条）。

一、**个人业绩考核**  
涵盖以下三类业务，且各项有明确的最低考核标准及相互折算规则：  
- **储蓄业务**（季日均余额）：为各类个金客户经理考核进入的最低标准；  
- **卡业务**（季新增有效发卡量）：为见习、D类、初级客户经理进入的最低标准；“有效卡”指每张卡月均余额≥100元；  
- **个贷业务**（季新增发放个贷）：为中级以上客户经理考核进入的最低标准。  

各项业绩可按 **50万元储蓄 = 50万元个贷 = 50张有效卡 = 5分** 的比例相互折算（来源：-4 说明第5条）。

二、**工作质量考核**（实行扣分制，满分隐含为100分，最多扣50分）：  
（1）**服务质量考核**（含7项）：  
- 工作责任心不强、缺乏协作精神：扣5分；  
- 客户服务效率低、态度生硬或有投诉：每次扣2分；  
- 不服从工作安排、不参加宣传活动：每次扣2分；  
- 未按时参加培训、考试或专题活动：每次扣2分；  
- 未按规定开展贷前调查或贷后检查：每笔扣5分；  
- 未建立信贷台账及档案：每笔扣5分；  
- 发生不廉洁自律行为：每次扣50分。  

（2）**个人资产质量考核**：  
- 收息率：当季≥97%为合格，每降1个百分点扣2分；  
- 不良资产：零为合格，每超1个百分点扣1分；  
- 逾期贷款：  
  ▪ 单笔≤10万元且当季收回：扣1分；  
  ▪ 2笔以上累计≤20万元且当季收回：扣2分；  
  ▪ 累计＞20万元：扣4分（来源：-5 第九条）。

此外，考核结果直接影响行员级别与待遇：  
- 各项考核分值达某一档行员级别标准，下一季度即享受该级薪资（-8 第十五条）；  
- 年度业绩考核未达技术职务下限者，次年职务相应下调（-8 第十六条）；  
- 考核与行员等级挂钩，实行“每年考核浮动”（-1 第四条）。
