[改写后的查询]
    客户经理的考核标准是什么？

[答案]
------------------------------------------------------------
客户经理的考核标准分为两大类：**个人业绩考核**和**工作质量考核**（来源：百度文库 -7 第三条）。

1. **个人业绩考核**内容包括：  
   - 个人资产业务（如个贷业务，季新增发放个贷为中级以上客户经理进入最低标准）；  
   - 负债业务（如储蓄业务，季日均余额为各类个金客户经理考核进入的最低标准）；  
   - 卡业务（季新增发有效卡量，为见习、D类、初级客户经理进入最低标准；有效卡指月均余额≥100元）（来源：百度文库 -4）；  
   - 业绩可相互折算，标准为：50万元储蓄 = 50万元个贷 = 50张有效卡 = 5分（以5分为单位折算）（来源：百度文库 -4）。

2. **工作质量考核**实行扣分制，满分基础上最多扣50分，包括：  
   - **服务质量**：如责任心不强（扣5分）、客户投诉（每次扣2分）、不服从安排（每次扣2分）、未参加培训（每次扣2分）、未按规定开展贷前/贷后检查（每笔扣5分）、未建信贷台账（每笔扣5分）、不廉洁自律（每次扣50分）等；  
   - **个人资产质量**：  
     - 当季收息率低于97%，每降1个百分点扣2分；  
     - 不良资产超零，每超1个百分点扣1分；  
     - 跨月逾期：单笔≤10万元且当季收回扣1分；2笔以上累计≤20万元且当季收回扣2分；超20万元扣4分；  
     - 逾期超3个月，无论金额大小和笔数，扣10分（来源：百度文库 -5）。

此外，考核结果与行员级别及薪资待遇挂钩：各项考核分值总计达到某一档行员级别考核分值标准，即可在下一季度享受该级薪资标准；技术职务每年考核浮动，未达下限者次年下调（来源：百度文库 -8 第十五条、第十六条；-1 第四条）。

    客户经理的工作质量考核中，除了被投诉一次扣2分之外，还有哪些扣分项？

[答案]
------------------------------------------------------------
根据上下文，客户经理工作质量考核（第九条）中，除“有客户投诉的，每投诉一次扣2分”外，还有以下扣分项：

1. 工作责任心不强，缺乏配合协作精神；扣5分（第九条（一）第1款）  
2. 不服从支行工作安排，不认真参加分（支）行宣传活动的，每次扣2分（第九条（一）第3款）  
3. 未能及时参加分行（支行）组织的各种业务培训、考试和专题活动的，每次扣2分（第九条（一）第4款）  
4. 未按规定要求进行贷前调查、贷后检查工作的，每笔扣5分（第九条（一）第5款）  
5. 未建立信贷台帐资料及档案的，每笔扣5分（第九条（一）第6款）  
6. 在工作中有不廉洁自律情况的，每发现一次扣50分（第九条（一）第7款）  
7. 个人资产质量考核中：  
　- 当季收息率每低于97%一个百分点，扣2分（第九条（二））  
　- 不良资产每超一个百分点，扣1分（第九条（二））  
　- 发生跨月逾期，单笔≤10万元且当季收回，扣1分（第九条（二）A）  
　- 发生跨月逾期，2笔以上累计≤20万元且当季收回，扣2分；累计＞20万元，扣4分（第九条（二）B）  
　- 发生逾期超过3个月，无论金额大小和笔数，扣10分（补充条款C，见-6页）  

注：以上均出自《上海浦东发展银行西安分行个金客户经理管理考核暂行办法》第五章第九条及后续补充条款。
