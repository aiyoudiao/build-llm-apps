## Day 12 总结

主要是在「大模型底层数学原理与推荐系统」的基础算法实现上。深入探索了 **ALS 矩阵分解 (Alternating Least Squares)** 在推荐系统中的应用，以及 **SVD 奇异值分解 (Singular Value Decomposition)** 在图像压缩中的原理。通过纯 Python 手写线性代数库以及利用 NumPy/SciPy 进行降维重构，直观展示了矩阵分解技术和低秩近似的核心数学思想，这些思想也是大模型微调（LoRA）的重要理论基础。

### 记忆卡片

- **ALS (交替最小二乘法)**：一种常用于协同过滤推荐系统的矩阵分解算法。通过将用户-物品评分矩阵分解为用户隐因子矩阵和物品隐因子矩阵，交替固定其中一个并求解另一个的最小二乘解，从而预测用户对未评分物品的喜好。
- **SVD (奇异值分解)**：将任意实数矩阵分解为三个特定矩阵（左奇异向量、奇异值对角阵、右奇异向量转置）乘积的线性代数方法，常用于降维、去噪和数据压缩。
- **低秩近似 (Low-Rank Approximation)**：通过仅保留 SVD 分解中前 k 个最大的奇异值，构造出原矩阵的最优近似矩阵，在大幅减少数据存储量的同时保留主要特征。这也是大模型参数高效微调（如 LoRA）的核心数学原理之一。

### 理解

- **推荐系统核心算法 (`1.ALS矩阵分解推荐系统.py`)**：
  - 手写实现了一个基础的 **Matrix** 线性代数库，支持矩阵乘法、高斯-约旦消元法求逆等底层计算，帮助深入理解算法原理。
  - 基于该库实现了 **ALS 推荐模型**。通过读取历史评分数据（如 `ratings_small.csv`），预处理构建稀疏字典，交替迭代更新用户矩阵和物品矩阵，最终计算出预测评分并生成 Top-N 电影推荐列表。
- **图像压缩与大模型微调原理 (`2.SVD图像压缩与还原.py`)**：
  - 利用 `scipy.linalg.svd` 对本地图片（支持灰度或 RGB 多通道图像）进行奇异值分解处理。
  - 通过设定不同的奇异值保留个数（k 值如 5, 50, 500）重构图像矩阵，并利用 Matplotlib 对比展示不同压缩率下的视觉效果与均方误差（MSE）。
  - 直观揭示了通过丢弃较小奇异值来实现数据压缩和去噪的过程，深刻模拟了低秩矩阵近似（Low-Rank Approximation）在实际场景中的应用。
